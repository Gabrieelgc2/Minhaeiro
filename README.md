## âš™ï¸ Quick Installation Guide

To get Minhaeiro up and running on your local machine, follow these simple steps:

### Prerequisites
* Install **[Python](https://www.python.org/downloads/)**
* **VS Code** is highly recommended for development.

### Installation Steps
1.  **Clone the repository:**
    ```bash
    > git clone [https://github.com/Gabrieelgc2/Minhaeiro.git](https://github.com/Gabrieelgc2/Minhaeiro.git)
    ```
2.  **Access the directory:**
    ```bash
    > cd Minhaeiro
    ```
3.  **Install the `prettytable` library:**
    ```bash
    > pip install prettytable
    ```
4.  **Run the project:**
    ```bash
    > python minhaeiro.py
    ```
